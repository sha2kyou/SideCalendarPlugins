## 每日智慧语录插件

使用 AI 生成个性化智慧语录，每3小时自动生成一条随机风格的语录。

### 功能特性

- 使用大语言模型生成原创智慧语录
- 每3小时生成一条语录，全天共8条
- 随机选择励志、哲理、诗意、实用、幽默风格
- 自动跟随应用语言设置
- 智能缓存到当天结束，减少 API 调用
- 根据时间段、季节、日期生成相关内容
- 每个时间段都有独特的颜色
